# AI-Resume-Screening-System

An AI-powered resume screening and candidate ranking platform built using Machine Learning and FastAPI.

## Features
- Resume parsing and cleaning
- Skill extraction from resumes
- Candidate ranking system
- TF-IDF based resume matching
- Machine Learning based candidate scoring
- FastAPI backend
- Recruiter dashboard
- Candidate dashboard

## Technologies Used
- Python
- FastAPI
- Machine Learning
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript

## Project Workflow
1. Resume data preprocessing
2. Resume cleaning
3. Skill extraction
4. Resume-job matching
5. Candidate ranking
6. Final score generation

## Important Files
- `resume_api.py` → Main FastAPI backend
- `train_model.py` → Model training
- `04_resume_matcher.py` → Resume matching logic
- `05_candidate_ranker.py` → Candidate ranking system
